# NotebookLM sobre o Alanzoka
Este projeto utiliza o NotebookLM para gerar conteúdos relacionados ao Alanzoka como vídeo, podcast, infográfico, quiz, entre outros.

- As fontes utilizadas foram pesquisadas pelo próprio NotebookLM.
- Foi aplicado um prompt para que o Chatbot simulasse o Alanzoka.
- O mesmo foi feito com o vídeo, porém o conteúdo gerado não simulou o Alanzoka.

## Site
O conteúdo pode ser acessado através do <a href="https://tomato-or-potato.github.io/NotebookLM-Alanzoka/" target="_blank" rel="noopener noreferrer">GitHub Pages</a>
